# F2Tools (preview)


This extension is the main software component of the ***super-lightweight, yaml document based project management environment (not only) for programmers*** required to efficiently manage e.g. software development projects in the way we do it at this initial phase of "[The Forty2 Project](tf2p.org)", up until The F2Tool - our main "product" - reaches maturity (other parts are Git, VS Code and another VS Code extension, [Custom Coloring](https://marketplace.visualstudio.com/items?itemName=altrue.CustomColoring)).

## Project Management in The Forty2 Project way

### Preamble

Even though there are miriads of free or cheap (at least for small teams) project management software (like JIRA) and we even have access to the heavy-weight ones (like codebeamer), we've found that managing our projects is way more efficient using these methods and this environment, even at the early stages when we lacked most of the functionalities of this extension.

If you're interested to see how it works in action - e.g. you have your own software project with not more, than 5 developers - please pay a visit to our [Deep Dive section](https://deepdive.tf2p.org); although at the time of writing we're still just preparing the documentation and how-tos and fixing the most annoying bugs in the extension, you can watch how we use it (and in general, how does this management environment work) on Twitch, dig into the details and if you are even more interested, by signaling your interest with sending us some message we have one more reason to speed up the delivery.
